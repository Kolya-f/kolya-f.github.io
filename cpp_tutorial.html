Вступ

C++ — це потужна мова програмування, яка широко використовується для розробки програмного забезпечення, ігор, операційних систем та інших високопродуктивних додатків. Вона поєднує в собі можливості низькорівневого програмування з високорівневими абстракціями. C++ був створений Б'ярном Страуструпом у 1980-х роках як розширення мови C.

Основні переваги C++ включають високу продуктивність, гнучкість та підтримку об'єктно-орієнтованого програмування. Ця мова є однією з найпопулярніших у світі програмування, і її вивчення є важливим кроком для кожного програміста.

Основи C++

Основи C++ включають розуміння таких понять, як змінні, типи даних, оператори, цикли та умовні оператори. Ось приклад простого коду на C++:


#include <iostream>
using namespace std;

int main() {
    cout << "Hello, World!" << endl;
    return 0;
}

У цьому прикладі ми використовуємо бібліотеку ``, яка дозволяє працювати з введенням і виведенням даних. Функція `main()` є точкою входу в програму, і саме тут починається виконання коду.

Змінні в C++ оголошуються за допомогою типів даних, таких як `int`, `float`, `double`, `char` тощо. Наприклад:


int number = 10;
float pi = 3.14;
char letter = 'A';

Цикли, такі як `for`, `while` і `do-while`, дозволяють повторювати певні дії. Умовні оператори, такі як `if`, `else` і `switch`, дозволяють керувати потоком виконання програми.

Об'єктно-орієнтоване програмування

C++ підтримує об'єктно-орієнтоване програмування (ООП), яке включає такі концепції, як класи, об'єкти, наслідування, поліморфізм та інкапсуляція. Ось приклад простого класу на C++:


class MyClass {
public:
    MyClass() {
        cout << "Об'єкт створено" << endl;
    }
    ~MyClass() {
        cout << "Об'єкт знищено" << endl;
    }
};

У цьому прикладі ми створили клас `MyClass` з конструктором і деструктором. Конструктор викликається при створенні об'єкта, а деструктор — при його знищенні.

Наслідування дозволяє створювати нові класи на основі існуючих. Наприклад:


class Animal {
public:
    void eat() {
        cout << "Я їм" << endl;
    }
};

class Dog : public Animal {
public:
    void bark() {
        cout << "Гав-гав!" << endl;
    }
};

Тут клас `Dog` успадковує методи класу `Animal`.

Стандартна бібліотека шаблонів (STL)

Стандартна бібліотека шаблонів (STL) є однією з найпотужніших частин C++. Вона містить шаблони для роботи з контейнерами, алгоритмами та ітераторами. Ось приклад використання вектора:


#include <vector>
#include <iostream>

int main() {
    std::vector<int> numbers = {1, 2, 3, 4, 5};
    for (int num : numbers) {
        std::cout << num << " ";
    }
    return 0;
}

У цьому прикладі ми використовуємо контейнер `vector`, який дозволяє зберігати набір елементів. STL також включає такі контейнери, як `list`, `map`, `set` тощо.
